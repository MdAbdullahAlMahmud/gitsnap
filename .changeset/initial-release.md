Initial release of gitcard — a CLI tool that generates beautiful visual snapshot cards of git repositories.

Features:
- Language breakdown bar with GitHub-style colors
- GitHub stats (stars, forks, issues, contributors) via API
- File count, repo size, total commits, repo age
- Recent commit history
- Topics/tags display
- Dark and light themes
- PNG, PDF, and SVG output formats
- 1200×630 default (og:image standard)
